# 🏆 SkillHub

A full-stack learning goal tracker. Built with:
- HTML, CSS, JS (frontend)
- Node.js + Express (backend)
- PostgreSQL or MongoDB (database)

## 🚀 Features
- User login/register
- Skill tracking with progress
- Clean, responsive UI with Bootstrap

## 🛠️ Setup
```bash
npm install
node server.js
